## 🎁 To Customize

**Update Your Info:**
- Profile name: HeroSection.tsx (line ~75)
- Bio text: HeroSection.tsx (line ~85)
- Social links: HeroSection.tsx (line ~100)
- Email: ContactSection.tsx
- Projects: ProjectsSection.tsx

**Add Images:**
- Profile pic: Replace `/api/placeholder/400/500`
- Project images: Add to `/public/images/`
- Logo: Update in Header.tsx

**Change Colors:**
- Edit `src/app/globals.css`
- Update Tailwind colors
- Change gradient values
