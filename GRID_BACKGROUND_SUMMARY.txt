================================================================================
                           PROP REFERENCE
================================================================================

dotSpacing (number, default: 25)
  → Distance between dots in pixels
  → Recommended: 15-35

dotSize (number, default: 1.5)
  → Size of dots in pixels
  → Recommended: 1-3

dotColor (string, default: #7C3AED)
  → Hex color of dots
  → Example: #7C3AED, #6366F1, #A78BFA

backgroundColor (string, default: #020617)
  → Main background color
  → Example: #020617, #0B0220, #F8F9FC

opacity (number, default: 0.4)
  → Opacity of dots (0-1)
  → 0 = invisible, 1 = fully opaque

enableGlow (boolean, default: true)
  → Add subtle glow effect to dots
  → Creates softer appearance

enableAnimation (boolean, default: false)
  → Enable subtle floating animation
  → 20s duration, low intensity

className (string, default: '')
  → Additional CSS classes

================================================================================
                      Z-INDEX AND POSITIONING
================================================================================

IMPORTANT: Z-Index Hierarchy

-z-10:  Grid Background (behind everything)
z-0:    Main content
z-50:   Header/Sticky elements
z-100+: Modals/Dropdowns

The grid uses "fixed inset-0 -z-10" to position itself correctly.
Ensure your content has proper z-index to appear above the grid.

Example:
<div className="relative w-full min-h-screen">
  <GridBackgroundCSS />
  <div className="relative z-0">
    {/* Your content here */}
  </div>
</div>

================================================================================
                       TROUBLESHOOTING
================================================================================

Q: Grid not showing?
A: Check z-index. Grid needs "-z-10", content needs "z-0" or higher.

Q: Grid affecting text readability?
A: Lower the opacity or increase backgroundColor contrast.

Q: Performance issues?
A: Use GridBackgroundCSS (CSS version) instead of GridBackground (Canvas).

Q: Looks different on mobile?
A: Adjust dotSpacing for different breakpoints or use responsive values.

Q: Animation causing issues?
A: Set enableAnimation={false} (it's disabled by default).
